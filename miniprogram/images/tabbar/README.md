# TabBar 图标说明

由于小程序需要本地图标文件，请按照以下方式添加图标：

## 图标要求
- 尺寸：81px × 81px
- 格式：PNG
- 颜色：未选中状态使用灰色 (#7A7E83)，选中状态使用主题色 (#667eea)

## 需要的图标文件

1. **home.png / home-active.png** - 首页图标
   - 建议使用：🏠 或 home 图标

2. **gallery.png / gallery-active.png** - 作品图标
   - 建议使用：🖼️ 或 image 图标

3. **ar.png / ar-active.png** - AR识物图标
   - 建议使用：📷 或 camera 图标

4. **shop.png / shop-active.png** - 商城图标
   - 建议使用：🛍️ 或 shopping 图标

5. **profile.png / profile-active.png** - 我的图标
   - 建议使用：👤 或 user 图标

## 临时解决方案

如果暂时没有图标文件，可以：
1. 使用在线图标生成工具生成
2. 从 iconfont 下载对应图标
3. 暂时移除 iconPath 配置，只使用文字（已配置）

## 图标资源推荐
- https://www.iconfont.cn/ (阿里巴巴图标库)
- https://iconify.design/ (Iconify 图标库)
